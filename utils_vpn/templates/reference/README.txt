Put your reference images here.

1. When you see the Cloudflare page, take a screenshot.
2. Crop ONE of these and save as PNG in this folder:
   - The "Verifying..." spinner + Cloudflare logo (small box)
   - The "Verify you are human" checkbox + Cloudflare (small box)
3. Any .png file in this folder will be searched for in the full-screen screenshot;
   when found, the script clicks the center of that match.

Use descriptive names, e.g. cloudflare_verifying.png or cloudflare_checkbox.png.
